## How to add page to the book

1. Add new .md file to  `src` folder, e.g `01_example.md`
2. Add new item to the list in [`SUMMARY.md`](./SUMMARY.md), e.g
`- [Example Page](./01_example.md)`
3. Push your changes and enjoy the result :)
